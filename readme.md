# Hypixel Skyblock Flipper Tool

A Python-based tool to track and visualize Hypixel Skyblock bazaar prices and find flipping opportunities using the official Hypixel API.

## Features
* **Real-time Data**: Fetches the latest bazaar prices directly from Hypixel.
* **Visual Analytics**: Generates price charts using Matplotlib to spot trends.
* **Watchlist**: Track specific items you are interested in via `watchlist.txt`.

## Prerequisites
* Python 3.8+
* A Hypixel API Key (Get one on Developer.hypixel.net).
